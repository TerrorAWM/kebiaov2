# 版本历史 (Version History)

> 仅记录 Release 版本，Beta 版本不在此列出。

## v2.1.2 (2026-04-02)

**周导航、分享权限与主题适配 (Week Navigation, Share Permission & Theme Support)**

- **新增功能**
  - 主课表、实验课表、分享页新增周导航（上一周/下一周）、周区间显示（`M.D - M.D`）、三周预计算缓存与连续切换。
  - “显示全部课程”模式下，点击周箭头会自动切回“仅本周”并切换周。
  - 全站新增主题模式：`自动 / 亮色 / 深色`，支持跨页面缓存；主课表/实验课表新增主题选择器，登录页同步生效。
- **优化与修复**
  - 分享页未授权跨周时强制仅当前周，避免跨周数据暴露。
  - `migrate.php` 改为使用 BACKUP 密码登录后才可进入。
  - 课表左侧表头移除“时间 / 节次”，桌面端调整为“第 X 节”在上、“时间段”在下。
  - 手机端节次列改为四行紧凑布局（节次数字、开始时间、`-`、结束时间）。
  - 课程胶囊移除 `cap-dot`，并优化手机端与深色模式显示。
- **数据库变更**
  - `shared_links` 表新增 `allow_week_nav` 字段（默认 `0`）。
  - 分享管理新增 `share_update`，支持按链接实时更新跨周权限。
  - `migrate.php` 增加旧库补字段与受控修复流程。

## v2.1.1 (2026-04-01)

**Bug 修复 (Bug Fixes)**

- **共享页数据库前缀修复**：修复 `share_kb.php` 在读取课表时直接使用 `user_schedule / user_lab_schedule` 导致的表不存在问题；现统一通过 `table()` 读取带前缀表名（如 `kb_user_schedule`），兼容 `config.php` 中的 `DB_PREFIX` 配置。

## v2.1.0 (2025-12-01)

**邮箱找回与密码重置 (Email Recovery & Password Reset)**

- **新增功能**
  - **账号找回**：新增 `findaccount.php`，支持通过邮箱查找账号并发送重置链接。
  - **密码重置**：新增 `reset.php`，支持通过安全令牌重置 PIN 码。
  - **邮箱管理**：在 `edit.php` 设置中支持绑定和修改邮箱。
  - **安装向导**：新增邮件配置步骤（Step 3），支持 SMTP 设置与测试。
- **数据库变更**
  - `user_accounts` 表新增 `email` 字段。
  - 新增 `password_resets` 表用于存储重置令牌。
  - 提供 `migrate.php` 用于旧版本数据库升级。

## v2.0.2 (2025-11-28)

**本次更新主要集中在功能优化与 Bug 修复。**

- **新增功能**
  - **实验课表按需显示**：未注册实验课表的用户将自动隐藏入口，且访问相关页面会自动重定向至注册页。
  - **显示字段自定义**：用户可自由选择在课表中显示哪些字段（课程名、教师、教室、周数）。
  - **设置菜单**：右侧齿轮图标升级为下拉菜单，提供“编辑”和“设置”的快捷入口。
- **修复与优化**
  - 修复了显示字段设置无法保存的问题。
  - 修复了 MariaDB 数据库的兼容性问题。
  - 优化了前端渲染逻辑，确保隐藏字段真正不可见。

## v2.0.1 (2025-11-25)

**重构版本 (Refactor)**

- **核心重构**：代码库全面重构，提升性能与可维护性。
- **新 UI**：引入全新的现代化用户界面。
- **核心修复**：彻底修复了旧版本中存在的时间与时区计算 等诸多问题。
