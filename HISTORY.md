# 版本历史 (Version History)

> 仅记录 Release 版本，Beta 版本不在此列出。

## v2.0.2 (2025-11-28)

**本次更新主要集中在功能优化与 Bug 修复。**

- **新增功能**
  - **实验课表按需显示**：未注册实验课表的用户将自动隐藏入口，且访问相关页面会自动重定向至注册页。
  - **显示字段自定义**：用户可自由选择在课表中显示哪些字段（课程名、教师、教室、周数）。
  - **设置菜单**：右侧齿轮图标升级为下拉菜单，提供“编辑”和“设置”的快捷入口。
- **修复与优化**
  - 修复了显示字段设置无法保存的问题。
  - 修复了 MariaDB 数据库的兼容性问题。
  - 优化了前端渲染逻辑，确保隐藏字段真正不可见。

## v2.0.1 (2025-11-25)

**重构版本 (Refactor)**

- **核心重构**：代码库全面重构，提升性能与可维护性。
- **新 UI**：引入全新的现代化用户界面。
- **核心修复**：彻底修复了旧版本中存在的时间与时区计算 等诸多问题。

## v1.0.0 (2024-09)

**双表系统引入**

- **双课表支持**：正式引入“主课表 + 实验课表”双系统架构。
- **功能增强**：新增课表分享功能，支持密码保护与有效期设置。
- **UI 改进**：界面初步优化。

## v0.1.0 (2023-11)

**初始版本**

- 基础课表功能发布。
